# NeuCard AI Demo Apps

Two Streamlit apps:

1) Cohort Intelligence + Feedback Loop
   - File: cohort_feedback_app.py
   - Data: mock_customer_data_extended.csv (upload in app or place in same folder)
   - Run: streamlit run cohort_feedback_app.py

2) Dynamic SEO Engine
   - File: dynamic_seo_app.py
   - Run: streamlit run dynamic_seo_app.py

## Deploy to Streamlit Community Cloud
- Create a public GitHub repo and upload:
  - cohort_feedback_app.py
  - dynamic_seo_app.py
  - mock_customer_data_extended.csv
  - requirements.txt
- Go to https://share.streamlit.io, connect your repo, choose the app file, and deploy.
